# references/

**Purpose:** Literature references, dataset documentation papers, and citation management for the manuscript.

## Suggested Contents

- `bibliography.bib` — Master BibTeX file for all cited works (used when writing the manuscript in `reports/paper/`).
- `papers/` — PDF copies of key reference papers (only where redistribution is permitted; otherwise store DOIs/links only).
- `dataset_papers/` — Original papers describing each benchmark dataset used (e.g., CIC-IDS2017 dataset paper, UNSW-NB15 dataset paper), important for correct citation and methodology description.

## Citation Management Recommendation

Use a reference manager (Zotero, Mendeley, or JabRef) synced to export a consolidated `bibliography.bib` here, ensuring consistent citation keys across `docs/research_notes.md` and `reports/paper/manuscript.tex`.

## Naming Convention

```
<FirstAuthorLastName><Year>_<ShortTitle>.pdf
```

Example: `Sharafaldin2018_CICIDS2017.pdf`
